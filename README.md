# 💰 Monthly Budget Tracker

A simple, private budget tracker. Set a monthly budget for each spending
category, log expenses as you go, see what's left, and share your remaining
balance with a tap.

Everything runs in a single file (`index.html`). No accounts, no internet
required, and all data stays **privately on your own device**.

## How to use it

1. **Open `index.html`** in any web browser (phone or computer).
2. Tap **Manage** to set up your categories and monthly budgets
   (e.g. Groceries $400, Clothes $150, Fun & dining $100).
3. Tap **＋ Add expense** whenever you spend, and pick the category.
4. Watch each category's **remaining balance** update, with a color bar
   (green → yellow → red as you approach the limit).
5. Tap **📤 Share my remaining** to send a tidy summary — it opens your
   phone's share sheet, or copies the text so you can paste it into a message.

The budget **resets automatically** on the 1st of each month: your categories
and budgets stay, and the expense list starts fresh. The header always shows
how many days are left before the reset.

## 📱 Tip: keep it on her phone

For the easiest everyday use, open the file once in the phone's browser and
**Add to Home Screen** — it then opens like an app, full-screen, and her data
stays saved between visits.

> Want a tap-to-open web link instead of a file? This repo can be published
> with **GitHub Pages** (Settings → Pages → deploy from the `main` branch),
> which gives you a private URL that works the same way on any device.
